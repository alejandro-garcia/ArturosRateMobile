## Install 📋

```bash

make copy of environment.example.ts  to 
./environments/environment.prod.ts 
./environments/environment.ts

$ npm install
$ npm run build
$ npx cap copy 
$ npx cap open android

after build from .apk files from android studio.
```

## Running the app

```bash
# development
$ ionic serve
```
